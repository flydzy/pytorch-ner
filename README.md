# 单语言命名实体识别
目的是为了对比不同的模型在英语上的命名实体识别的效果，同时巩固一下自己的深度学习的技能。涉及到的模型有
1. HMM
2. CRF
3. LSTM
4. BiLSTM
5. BiLSTM_CRF
6. BERT
7. BERT_CRF

使用到的技术有
1. pytorch
2. torchtext

# 实验结果记录F1值
